Projet JEE/EJB
Marine Doyen-Le Boulaire

Le projet suivant est un Java Enterprise Project. Il est composé de trois projets : dvdEnterprise-ejb-dao-jpa en charge de la gestion de la base de données proprement dite, dvdEnterprise-ejb-metier qui implémente la couche métier et dvdEnterprise-web qui se charge de l'affichage.

La page principale de l'application web est index.jsp.

Les fichiers de création et d'initialisation de la BD sont createBD.sdql et initBD.sql dans dvdEnterprise-ejb-dao-jpa/src/java.sql.
